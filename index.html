Peta berbasis WEB ini disederhanakan dari Bootleaf Template bmcbride Dengan memanfaatkan perpustakaan Bootstrap , Leaflet dan typeahead.js. Sehingga aplikasi ini tidak berbayar sesuai dengan prinsip Open source dan MIT licensed.

Sekilas Tentang WebGIS Banjir
  • WebGIS Banjir ini dibangun untuk tahapan lanjutan dari interview dalam proses rekrutmen GIS Techical di Perkumpulan OpenStreetMap Indonesia. saya terinspirasi membuat WebGIS Banjir untuk di Kota Pontianak berdasarkan data bangunan dari OpenStreetMap, yang saya buat menjadi gridmap untuk meminimalisir loading pada webgis. Kemudian data LiDAR Kota Pontianak saya dapatkan dari Bappeda Provinsi Kalimantan Barat untuk membuat simulasi banjir genangan pasang surut dan hujan. Data Risiko Banjir ini hasil analisa dari Kepadatan Bangunan yang di overlay dengan data Rawan Banjir Genangan
